# Violence District | Smart-ESP

**Проект полностью Open-Source**

### Основные возможности

- **Раздельная подсветка состояний выживших:**
  - Здоровые Survivors — ярко-зелёный
  - Раненые — оранжевая обводка
  - Актвина Неуязвимость — бирюзовый цвет + синяя обводка
- **Убийца** — подсвечивается красным цветом
  - Добавлено отображение количества повешаний как у Survivors 
- **Генераторы:**
  - Обычные — белый контур (По мере починки зеленеет)
  - Полностью отремонтированные — невидимый

- Максимальная оптимизация кода
