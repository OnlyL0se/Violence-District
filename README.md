# Violence District | Smart-ESP

## Проект полностью Open-Source

### Что делает скрипт

Скрипт автоматически подсвечивает всех игроков и генераторы через Highlight, меняя цвет и стиль в зависимости от текущего состояния.

### Основные возможности и фишки

- **Плавные цветовые переходы** — при смене состояния (здоровый → раненый, раненый → неуязвимый и т.д.) цвет Highlight плавно анимируется через TweenService.

- **Раздельная подсветка состояний выживших:**
  - Здоровые Survivors — ярко-зелёный
  - Раненые (Hurted/Injured, хромают) — зелёный Fill + оранжевый Outline
  - С включёнными Iframes — бирюзовый цвет с синим контуром

- **Killer** подсвечивается красным цветом.

- **Генераторы:**
  - Обычные — белый контур
  - Полностью отремонтированные — ярко-зелёный контур

- Автоматическое определение раненого состояния
