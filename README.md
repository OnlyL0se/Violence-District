# Violence District | Smart-ESP

**Проект полностью Open-Source**

### Что делает скрипт

Скрипт автоматически подсвечивает всех игроков и генераторы через **Highlight**, динамически меняя цвет и стиль в зависимости от текущего состояния персонажа или объекта.

### Основные возможности и фишки

- **Плавные цветовые переходы** — при смене состояния цвет Highlight плавно анимируется через TweenService (без резких скачков).
- **Раздельная подсветка состояний выживших:**
  - Здоровые Survivors — ярко-зелёный
  - Раненые (Hurted / Injured, хромают) — зелёный Fill + оранжевый Outline
  - Активные Iframes — бирюзовый Fill + синий Outline
- **Killer** — подсвечивается красным цветом
- **Генераторы:**
  - Обычные — белый контур
  - Полностью отремонтированные — невидемый

- Автоматическое определение раненого состояния
- Максимальная оптимизация кода

Скрипт лёгкий, стабильный и продолжает активно улучшаться.
